# Gestion de Garage Automobile

Application web complète de gestion de garage développée en JEE.

## Fonctionnalités
- Gestion des clients, véhicules et réparations
- Accès basé sur les rôles
- Facturation automatisée (calcul des coûts via EJB et JPA)
- Génération de rapports PDF
- Historique complet des interventions

## Technologies
- **Backend :** Java, JEE (EJB, JPA, CDI)
- **Frontend :** JSP/JSF
- **Base de données :** MySQL
- **Serveur :** WildFly
- **Architecture :** MVC en couches
